약봉투 전체 OCR → 일괄 DUR 체크

킬러 기능

📖

의약 용어 쉬운 말 변환

자체 의약 용어 사전으로 어려운 표현을 쉬운 말로 변환. "간대사" → "간에서 분해됩니다". Phase 2에서 LLM 연동으로 고도화 예정.

🛡️

소비자 친화적 병용금기 체크

DUR API 데이터를 기반으로 일반인도 이해하기 쉬운 방식으로 약물 상호작용 정보를 제공. 약국/병원에 가지 않아도 확인 가능.

🏛️

공식 공공 데이터 기반 신뢰성

식약처·DUR 공식 API 사용으로 정확도 보장. 개인 블로그나 커뮤니티 정보와 달리 검증된 의약 정보를 제공.
